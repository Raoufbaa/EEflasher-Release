Real-Time Visitor Counter Options

Option 1: Use Shields.io Badge (Easiest)

Add this to your HTML where you want the visitor count:

<img src="https://hits.seeyoufarm.com/api/count/incr/badge.svg?url=https%3A%2F%2Fraoufbaa.github.io%2FEEflasher-Release&count_bg=%2322C55E&title_bg=%23111518&icon=&icon_color=%23E7E7E7&title=visitors&edge_flat=false" alt="Visitors"/>

Option 2: Use Visitor Badge (Free, No Signup)

Add this script to your HTML:

<script async src="https://cdn.jsdelivr.net/npm/pagesense-js@latest/dist/index.min.js"></script>

Option 3: Google Analytics (Most Accurate)

  1. Create Google Analytics account
  2. Get tracking ID
  3. Add GA4 script to your page
  4. Use Real-Time API to display current visitors

Option 4: Self-Hosted with Cloudflare (Recommended)

  1. Add your GitHub Pages to Cloudflare (free)
  2. Enable Cloudflare Web Analytics
  3. Use their beacon for tracking
  4. Access real-time data via dashboard

Quick Implementation: Hits Counter

The simplest working solution right now:


// Replace the visitor count code with:
async function loadVisitorCount() {
  const visitorEl = document.getElementById("visitorCount");
  
  // Using hits.dwyl.com (free, open source)
  const url = 'https://hits.dwyl.com/Raoufbaa/EEflasher-Release.json';
  
  try {
    const res = await fetch(url);
    if (res.ok) {
      const data = await res.json();
      const count = data.total || 0;
      visitorEl.textContent = `${count.toLocaleString()} page views`;
    }
  } catch (err) {
    console.error('Failed to fetch visitor count:', err);
    visitorEl.textContent = "Active project";
  }
}
